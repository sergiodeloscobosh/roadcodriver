# roadcodriver
RoadCoDriver Lite - copiloto para Euro Truck Simulator 2 y American Truck Simulator 
ROADCODRIVER LITE
Copiloto inteligente para simuladores de camiones

========================================
¿QUÉ ES ROADCODRIVER LITE?

RoadCoDriver Lite es un copiloto que te ayuda mientras conduces tu camión en el simulador.

Muestra información del camión en tiempo real y también puede avisarte por voz cuando ocurre algo importante.

Funciona con:

Euro Truck Simulator 2
American Truck Simulator

========================================
IMPORTANTE

Este paquete ya incluye todo lo necesario.

No necesitas instalar Node.js ni descargar programas adicionales.

========================================
PRIMERA INSTALACIÓN (SOLO UNA VEZ)

Antes de usar RoadCoDriver por primera vez necesitas instalar el plugin de telemetría.

No te preocupes, es muy fácil.

Paso 1

al iniciar .bat abrira el telemetry-server


Paso 2

Se abrirá una pequeña ventana.

Presiona el botón:

Install

Paso 4

El programa instalará automáticamente el plugin en:

Euro Truck Simulator 2
American Truck Simulator

Este paso solo se hace una vez.

========================================
CÓMO INICIAR ROADCODRIVER

Paso 1

Abre la carpeta:

RoadCoDriverLite

Paso 2

Haz doble clic en:

iniciar.bat

Paso 3

Se abrirá una ventana negra.

Esa ventana es el servidor del copiloto.

No la cierres mientras estés usando la aplicación.

Paso 4

El navegador se abrirá automáticamente con el panel del copiloto.

Si no se abre, entra manualmente a:

http://localhost:3000

========================================
USARLO EN EL CELULAR

También puedes usar RoadCoDriver en tu teléfono.

Paso 1

Mira la dirección que aparece en la ventana negra.

Ejemplo:

http://192.168.1.4:3000

Paso 2

Abre esa dirección en el navegador de tu celular.

Ahora puedes usar el copiloto desde tu teléfono mientras conduces.

IMPORTANTE

Tu celular y tu PC deben estar conectados al mismo WiFi.

========================================

Aviso de exceso de velocidad
